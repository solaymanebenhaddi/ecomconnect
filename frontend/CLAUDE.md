# Frontend — EcomConnect
## Responsibility
The frontend layer owns the user experience, administrative dashboards, and data visualization for the e-commerce connectivity platform.

## Entry points
- Main entry: `frontend/src/app/page.tsx`
- Run command: `npm run dev`

## Key conventions
- **Naming**: camelCase for variables/functions, PascalCase for components.
- **Structure**: App router architecture (Next.js).
- **Testing**: Jest and Playwright.

## Do NOT touch
- `package-lock.json` manually.
- Global style overrides in `globals.css` without coordination.
